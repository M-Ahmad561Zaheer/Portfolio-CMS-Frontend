# 🚀 Portfolio CMS Frontend

A modern, responsive, and SEO-friendly Portfolio CMS frontend built with **React**, **Vite**, **Tailwind CSS**, and **Axios**. The application integrates seamlessly with a secure ASP.NET Core backend to provide a complete portfolio management experience.

# 🚀 Getting Started

Clone repository

```bash
git clone https://github.com/M-Ahmad561Zaheer/Portfolio-CMS-Frontend.git
```

Navigate to project

```bash
cd Portfolio-CMS-Frontend
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
